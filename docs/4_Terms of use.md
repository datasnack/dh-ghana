# Terms of Use


### Data provided by the Data Hub

Each dataset in the **Data Hub** is hosted under its own license, ensuring compliance with the intellectual property rights of its original source. Before using any dataset, review the terms in its metadata — these specify attribution requirements, permitted uses, and copyright information. By accessing and using data through the Data Hub, users agree to comply with the applicable license terms, including crediting the data custodian in the manner specified and complying with relevant copyright provisions.

<br>

### Metadata

Metadata documentation for all Data Layers is available via the [Data Layers overview]({% url 'datalayers:datalayer_index' %}), including detailed descriptions, attribution requirements, and usage terms. We are committed to the [FAIR principles](https://www.go-fair.org/fair-principles/) and currently follow metadata standards established by [DataCite](https://datacite.org/), and domain-specific guidelines for geospatial data. Our ongoing work focuses on improving metadata clarity and accessibility to better support [cross-domain collaboration](https://zenodo.org/records/11236871). Within the **Data Hub**, metadata is maintained by the project teams to ensure that project partners and third parties can gain a transparent understanding of the datasets — thereby strengthening data interoperability.

Please note that metadata provided in this demo environment is illustrative and may not be complete or fully accurate. It is shared without warranty and for demonstration purposes only.
